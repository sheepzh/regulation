# 相关法律法规

一款浏览器扩展。自定义相关法律法规，自动将网页里你不想看见的敏感/违禁词替换成安全词。

## 开发

### 启动运行环境

```nodejs
npm install
npm run dev
```

然后在浏览器开发模式下将 ./dist_dev 导入

### 单元测试

```nodejs
npm run test
# 或者
 npm run test-c
```

## 编译

```nodejs
npm install
npm run build
```

输出目录为 dist_prod
